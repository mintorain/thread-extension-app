왜 ThreadHook인가요?

원클릭 생성

페이지 추출부터 스레드 생성, 클립보드 복사까지 버튼 하나로 완료됩니다.

4개 AI 모델

Claude, ChatGPT, Gemini, Grok 중 원하는 모델을 자유롭게 선택할 수 있습니다.

서버 설치 불필요

백엔드가 클라우드에 배포되어 있어 확장앱만 설치하면 바로 사용합니다.

자동 클립보드 복사

생성된 스레드가 자동으로 클립보드에 복사되어 바로 붙여넣기하면 됩니다.

톤 & 길이 조절

전문적/중립/캐주얼, 짧게/보통/길게 옵션으로 원하는 스타일로 생성합니다.

사이드 패널 UI

Chrome 사이드 패널에서 동작하여 페이지를 보면서 동시에 스레드를 생성합니다.

How it works
4단계로 끝나는 스레드 생성

확장앱 설치

ZIP 다운로드 후 Chrome에 로드합니다. 30초면 설치 완료.

API Key 등록

설정에서 Provider를 고르고 API Key를 붙여넣기합니다.

페이지 추출

뉴스/블로그 페이지에서 버튼 클릭으로 본문을 자동 추출합니다.

스레드 생성

"스레드 생성" 클릭! AI가 훅-본문-해시태그를 만들어 클립보드에 복사합니다.

Gemini는 무료, Claude/ChatGPT/Grok은 사용량 과금. 스레드 1회당 약 $0.001~0.01입니다.
