ThreadHook v0.3

뉴스를 AI 스레드로 한 번에

브라우저에서 보고 있는 뉴스/블로그를 클릭 한 번으로 SNS 스레드 초안으로 변환합니다.

Claude / ChatGPT / Gemini / Grok 서버 설치 불필요 클립보드 자동 복사

1) Chrome 확장앱 설치

1

확장 프로그램 페이지 열기

chrome://extensions
2

개발자 모드 활성화

우측 상단 개발자 모드 토글을 켭니다.

3

확장앱 폴더 로드

"압축해제된 확장 프로그램을 로드" 클릭 후 다운받은 thread-extension-app 폴더를 선택합니다.

2) 빠른 시작 가이드

  • 1확장앱 설치 (좌측 참고)
  • 2확장앱 아이콘 클릭 (사이드 패널 열림)
  • 3설정(⚙)에서 Provider 선택 + API Key 등록
  • 4뉴스 페이지에서 페이지 추출 클릭
  • 5Tone/Length 선택 후 스레드 생성
  • 6결과가 클립보드에 자동 복사됨
서버 설치가 필요 없습니다. 백엔드가 이미 클라우드에 배포되어 있습니다.

3) API Key 발급 안내

아래 Provider 중 하나의 API Key만 있으면 사용할 수 있습니다.

6) 자주 묻는 질문

Q

서버를 직접 실행해야 하나요?

아니요. 백엔드가 클라우드에 배포되어 있어 확장앱만 설치하면 바로 사용할 수 있습니다.

Q

API Key 비용이 드나요?

Gemini는 무료, Claude/ChatGPT/Grok은 사용량에 따라 소액 과금됩니다. 스레드 1회 생성에 약 $0.001~0.01입니다.

Q

API Key가 안전한가요?

키는 서버 메모리에만 임시 저장되며, 서버 재시작 시 초기화됩니다. 키를 영구 저장하지 않습니다.

7) 문제 해결

!

"페이지 추출 실패"

해당 탭을 새로고침하고 사이드 패널을 다시 열어 시도합니다. Chrome 내부 페이지(chrome://)에서는 추출이 불가합니다.

!

"API Key가 유효하지 않습니다"

Provider 콘솔에서 키를 재확인하세요. 키 복사 시 앞뒤 공백이 포함되지 않았는지 확인합니다.

!

"백엔드 연결 실패"

설정에서 백엔드 연결 테스트 버튼으로 확인합니다. 기본 URL을 변경하지 마세요.

확장앱 업데이트 후에는 chrome://extensions에서 새로고침 아이콘을 눌러주세요.
