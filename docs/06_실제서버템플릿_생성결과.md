## 5) 실행 방법
### FastAPI
```bash
cd backend-fastapi
python -m venv .venv
. .venv/Scripts/Activate.ps1
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

### NestJS
```bash
cd backend-nestjs
npm install
npm run start:dev
```

## 6) 주의사항
- 현재 키 저장은 데모용 인메모리 구조입니다.
- 운영 전환 시 DB + KMS 암호화 저장으로 교체해야 합니다.
- Adapter 내부는 더미 응답이며, 실제 API 호출 로직으로 교체해야 합니다.
