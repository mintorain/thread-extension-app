ThreadHook MVP

확장앱 설치 & 첫 실행 온보딩

아래 순서를 따라가면 설치부터 스레드 생성까지 한 번에 확인할 수 있습니다.

1) Chrome 확장앱 설치

1

확장 프로그램 페이지 열기

chrome://extensions
2

개발자 모드 활성화

우측 상단 개발자 모드 토글을 켭니다.

3

압축해제된 확장 프로그램 로드

개발자 모드를 켜면 상단에 "압축해제된 확장 프로그램을 로드" 버튼이 표시됩니다.

사용자가 압축을 푼 폴더에서 thread-extension-app 폴더를 선택하세요.

예시: C:\Users\사용자이름\Downloads\threadhook\thread-extension-app

2) 실행 전 체크리스트

  • !백엔드 실행: FastAPI(http://127.0.0.1:8000) 또는 NestJS(http://127.0.0.1:3000)
  • !확장 아이콘 클릭 후 Backend URL 입력
  • !Provider 선택 + API Key 저장
  • !페이지 추출 → 스레드 생성
  • !생성 완료 시 클립보드 자동복사 확인

3) 실제 화면 스크린샷

4) 자주 생기는 문제

A

"페이지 추출 실패"가 뜰 때

해당 탭을 새로고침하고 확장 팝업을 다시 열어 시도합니다.

B

"생성 실패"가 뜰 때

백엔드 서버 실행 여부, Backend URL, API Key 저장 성공 여부를 확인합니다.

C

일부 페이지에서 작동하지 않을 때

브라우저 내부 페이지는 콘텐츠 스크립트 제한으로 추출이 안 될 수 있습니다.

확장앱 파일을 수정한 뒤에는 chrome://extensions 에서 해당 확장의 "새로고침"을 눌러 변경사항을 반영하세요.
