ThreadHook MVP

설정

인증 페이지 승인 후 등록 상태가 자동 확인됩니다.

상세 매뉴얼

1) OAuth(어소) 인증 방법
  1. 설정(⚙) 아이콘 클릭
  2. Backend URL을 http://127.0.0.1:8000으로 설정
  3. Provider 선택
  4. 인증 방식에서 OAuth(어소) 선택
  5. 인증 페이지가 자동으로 열리면 승인 클릭
  6. 팝업 상태가 OAuth 등록 완료로 바뀌면 완료
2) API 등록 방법
  1. 설정(⚙) 아이콘 클릭
  2. Provider 선택
  3. 인증 방식에서 API 선택
  4. API Key 입력 후 API 등록 버튼 클릭
  5. 상태가 API Key 저장 완료로 표시되면 완료
3) 사용 방법
  1. 뉴스/블로그 페이지를 연 상태에서 확장앱 사이드바 열기
  2. 페이지 추출 클릭
  3. Tone/Length 선택
  4. 스레드 생성 클릭
  5. 결과가 자동으로 클립보드에 복사됨
