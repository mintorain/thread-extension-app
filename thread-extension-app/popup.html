설정

키 발급받기

상세 매뉴얼

1) API Key 등록 (필수)
  1. 설정(⚙) 아이콘 클릭
  2. Provider 선택 (claude / chatgpt / gemini / grok)
  3. 키 발급받기 링크를 눌러 Provider 사이트에서 API Key 복사
  4. API Key 입력 후 API 등록 클릭
  5. 상태가 API 등록 완료로 표시되면 완료
2) 스레드 생성 방법
  1. 뉴스/블로그 페이지를 연 상태에서 확장앱 아이콘 클릭
  2. 페이지 추출 클릭하여 본문 가져오기
  3. Tone(전문/중립/캐주얼), Length(짧게/보통/길게) 선택
  4. 스레드 생성 클릭 (AI 응답까지 수 초 소요)
  5. 결과가 자동으로 클립보드에 복사됨
3) API Key 발급처
  1. Claude: console.anthropic.com (sk-ant-...)
  2. ChatGPT: platform.openai.com/api-keys (sk-...)
  3. Gemini: aistudio.google.com/apikey
  4. Grok: console.x.ai (xai-...)
4) 문제 해결
  1. API Key 유효하지 않음: Provider 콘솔에서 키 재확인
  2. 백엔드 연결 실패: 연결 테스트 버튼으로 확인
  3. 페이지 추출 실패: 페이지 새로고침 후 재시도
  4. 서버 재시작 후 키 초기화: API Key 재등록 필요
